# ESLint Driver

Provides [ESLint](https://github.com/eslint/eslint) support for
[Beemo](https://github.com/milesj/beemo), a build tool manager.

## Installation

```
yarn add @beemo/driver-eslint eslint
// Or
npm install @beemo/driver-eslint eslint --save
```
